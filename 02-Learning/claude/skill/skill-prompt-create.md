
# 创建新 Skill: prompt

---

## Skill 基本信息

- **Skill 名称**: `prompt`
- **Skill 级别**: `user`（user = 用户级全局 Skill，project = 项目级 Skill）
- **功能描述**（做什么，什么时候触发）:

只有在用户手动输入/promt的时候才能出发,工具不能自己调用这个skill,需要能支持splash命令补全

## 输入要求

这个 Skill 需要什么输入:

用户输入了提示词的时候,你需要分析这个用户用这句提示词是为了解决什么问题,如果你分析出来有可能多种问题,就让用户选,选完之后,再询问用户有没有额外的信息,最后你要去优化用户输入的promt,告诉用户这个提示词应该怎么写,用户的提示词有什么问题,哪里需要改进

## 期望输出

期望输出是什么:

1. 经过优化的提示词
2. 用经过优化的提示词和用户的提示词作对比,分析用户的提示词的缺点,告诉用户应该怎么优化,为什么这样优化,最终的目的是为了帮助用户提升他的提示词的质量

## 需要遵守的规则

列出 Claude 必须遵守的具体规则:

1. 只能通过splash命令调用,ai自己不能调用,比如要用显式调用
2. 你需要先分析用户要解决的问题,判断出这个问题是啥,和用户确认这个问题,要确认清晰了之后,才能进行提示词优化
3. 当用户认可了你提供的提示词之后,再读取这个经过优化后的提示词再真正的执行任务

## Claude 绝对不能做的事

列出 Claude 绝对不能做的事:

在用户确认优化了之后的提示词之前,不能执行任何写操作,不能改变任何文件
